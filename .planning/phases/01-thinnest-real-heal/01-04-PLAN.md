<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| live DOM → fingerprint/candidate signals | Untrusted page content captured into signals |
| candidate uniqueSelector → page.locator() | DOM-derived string interpolated into a selector |
| worker → main | Heal events cross the process boundary via testInfo.attach |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | candidate uniqueSelector -> page.locator() | mitigate | Build selectors from controlled signal values; prefer test-id/nth over interpolated free text; quote/escape attribute values (ASVS V5) |
| T-04-02 | Information disclosure | capture.ts fingerprint | mitigate | Store normalized/derived signals only, never raw innerText/full DOM (ASVS V10/V14) |
| T-04-03 | Repudiation / integrity | heal loop false green | mitigate | Re-throw on no-fingerprint, no-candidate, below-floor; heal only above conservative floor (Pitfall 2); proven by no-false-green control test |
| T-04-04 | Denial of service | premature heal / flakiness | mitigate | Heal only on the real TimeoutError after auto-wait; bounded replay budget from FINDINGS (Pitfall 1, HEAL-02) |
| T-04-05 | Information disclosure | network in heal path | mitigate | Zero network calls anywhere in capture/score/decide/rebind (offline guarantee; full CI network-block proof is Phase 4) |
</threat_model>
